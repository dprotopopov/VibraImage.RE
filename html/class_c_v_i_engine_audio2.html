Detailed Description

Видимо реализация API для управления камерой и микрофоном. Вполне возможно что автор сперва сделал полиграф для голоса, а для видео потом расписал. А название осталось. Вероятнее всего просто запускается параллельная нить в Windows? которая отвечает за захват изображения и звука через стандартные API мультимедиа Windows, и выдаёт результат в программу по запросам от программы.
